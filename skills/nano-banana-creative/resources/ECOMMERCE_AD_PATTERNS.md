# E-commerce Ad Patterns for Nano Banana

| Ad Type | Lighting Strategy | Composition Tip |
| :--- | :--- | :--- |
| **UGC Style** | Natural, slightly overexposed | Handheld camera feel, 1st person POV |
| **Luxury** | Hard rim light, dark background | Center-weighted, high-contrast |
| **Tech/SaaS** | Bright, even "Apple-style" | Clean white/pastel backgrounds, floating elements |
| **Fashion** | Golden hour or Studio Softbox | Rule of thirds, dynamic movement |

### Text Placement Safety Zones

* **Top 15%**: Leave empty for Meta Profile icons.
* **Bottom 20%**: Leave empty for "Shop Now" buttons and captions.
